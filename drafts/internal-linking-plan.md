**Status:** DRAFT — For review before Webflow implementation

**Purpose:** Document all cross-links between the new Investor Services page and existing pages to ensure a coherent internal linking structure.

## Linking Principles

- Every link should feel contextually natural — not forced
- Link text should describe the destination clearly
- Avoid overlinking; each page should have 2–4 relevant cross-links, not 10+
- Internal links should support both user navigation and SEO topic clustering
