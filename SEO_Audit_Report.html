14. Webflow-Specific Implementation Guide

Where to Add Schema Markup in Webflow

Schema Type Location in Webflow
LocalBusiness (site-wide) Project Settings → Custom Code → Head Code
Paste the JSON-LD script. It will appear on every page.
FAQPage (specific page) Page Settings → Custom Code → Before </body> tag
Add the FAQPage JSON-LD script to just the FAQ page.
BlogPosting (CMS template) CMS Collection Page Settings → Custom Code → Before </body>
Use Webflow Embed elements with CMS field references to dynamically populate schema fields.
BreadcrumbList Page Settings → Custom Code for static pages, or use a Webflow Embed component in the page template.

Webflow SEO Settings Checklist

  • Project Settings → SEO: Enable auto-generated sitemap, set global meta title format
  • Each Page → Settings → SEO: Set unique title tag, meta description
  • Each Page → Settings → Open Graph: Set OG title, description, image
  • CMS Collection → SEO fields: Ensure each collection item has SEO title, description, OG image fields
  • Images: In the Asset Manager, add alt text to every uploaded image. For CMS images, add an "Alt Text" field to the collection.
  • 301 Redirects: Project Settings → Hosting → 301 Redirects (if you change any URLs)

Webflow Limitations to Be Aware Of

  • robots.txt: Webflow auto-generates this with limited customization. You can add custom rules via Project Settings → SEO.
  • Sitemap metadata: Webflow auto-generates the sitemap but lastmod only updates when you publish changes. priority and changefreq are not customizable.
  • llms.txt: Webflow doesn't natively support a /llms.txt route. Workaround: Create a static page or use a 301 redirect from a custom path. Alternatively, host the file separately.
  • Custom headers: Limited control over HTTP headers. Some advanced technical SEO (like security headers) may need Webflow Enterprise or a reverse proxy.
